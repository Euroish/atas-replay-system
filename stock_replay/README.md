# Stock Replay Development Environment

This directory contains the implementation workspace for the A-share order-flow replay system.

Installed local environment:

- `backend/`: Python backend dependency list.
- `.venv/`: local Python virtual environment.
- `frontend/`: Vite React TypeScript frontend, created after npm install.
- `data/`: local raw/processed/app data.
- `logs/`: import/replay/validation logs.
